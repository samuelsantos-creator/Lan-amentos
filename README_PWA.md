# Apontamento · Produção & Paradas (PWA)

Sistema moderno e intuitivo para registro de apontamentos industriais utilizando o **Modelo de Subtração**.

## ✨ Funcionalidades
- **Offline First**: Funciona sem internet através de Service Workers.
- **PWA**: Pode ser instalado no celular ou desktop como um aplicativo nativo.
- **Modelo Subtração**: Focado na eficiência, descontando paradas do tempo bruto.
- **Auto-preenchimento**: Agiliza o registro de matrículas, produtos e recursos.
- **Exportação**: Permite baixar os dados registrados em formato JSON.

## 🚀 Como Abrir
1. Basta abrir o arquivo `index.html` no seu navegador (Chrome ou Edge recomendado).

## 📲 Como Instalar (PWA)
### No Android (Chrome)
1. Abra o link no Chrome.
2. Toque nos três pontos (⋮) no canto superior direito.
3. Selecione **"Instalar aplicativo"** ou **"Adicionar à tela inicial"**.

### No iPhone (Safari)
1. Abra o link no Safari.
2. Toque no ícone de **Compartilhar** (quadrado com seta para cima).
3. Role para baixo e selecione **"Adicionar à Tela de Início"**.

### No Computador (Chrome/Edge)
1. Clique no ícone de instalação na barra de endereços (ao lado da estrela de favoritos).
2. Clique em **Instalar**.
